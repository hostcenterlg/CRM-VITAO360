   - Ler col C (col 3) de rows 4-537

   - FAIL se qualquer celula vazia

3. **Check 2: REDE-02 -- Zero #REF! na aba REDES_FRANQUIAS_v2**
   - Verificar que aba REDES_FRANQUIAS_v2 existe no workbook
   - FAIL IMEDIATO se aba nao existe

4. **Check 3: REDE-03 -- Sinaleiro de penetracao atualizado com dados 2025**
   - Na aba REDES_FRANQUIAS_v2:
     * Verificar que col N (SINALEIRO %) contem formulas (nao valores estaticos) para rows 4-23

     * Verificar que col O (COR) contem formulas IF para rows 4-23
     * Verificar que col L (FAT.POTENCIAL) contem formulas referenciando PARAMETROS ($W$4, $W$5)
   - Contar formulas total na aba (esperado: >= 200)
   - PASS se todas as verificacoes de formula passam
   - PASS_WITH_NOTES se formulas existem mas algum check secundario falha

5. **Check 4: REDE-04 -- Metas 6M por rede operacionais**
   - Na aba REDES_FRANQUIAS_v2:
     * Verificar que col M (META 6M) tem valores para as redes principais
     * Cross-check com valores esperados da pesquisa:
       - FITLAND: ~R$ 283.500
       - CIA DA SAUDE: ~R$ 351.000
       - VIDA LEVE: ~R$ 154.500
       - DIVINA TERRA: ~R$ 157.500
     * Verificar que col Q (GAP) contem formulas = META - FAT.REAL
   - PASS se metas preenchidas e GAP calculado
   - PASS_WITH_NOTES se valores diferem levemente da referencia (tolerancia +-5%)

6. **Check 5: Integridade V13 -- Formulas PROJECAO preservadas**
   - Contar formulas na aba PROJECAO (celulas que comecam com "=")
   - PASS se >= 19.224
   - FAIL se < 19.224
   - Verificar freeze_panes preservado
   - Verificar que todas as abas esperadas existem (PROJECAO, LOG, DASH, CARTEIRA, REDES_FRANQUIAS_v2, e DRAFT tabs)

7. **Check 6: Tabela de referencia AS:AZ expandida**
   - Verificar que tabela AS:AZ tem 19 redes + SEM GRUPO (20 rows de dados)
   - Verificar que VLOOKUPs em F:J referenciam range expandido (conter "AX$22" ou similar)
   - Spot-check: F4 contem VLOOKUP com range incluindo row 22+

8. **Check 7: Cross-check distribuicao de redes**
   - Contar clientes por rede em col C
   - Comparar com tabela AS:AZ (cada rede na tabela deve ter >= 1 cliente no PROJECAO, exceto redes exclusivamente SAP)
   - Verificar que nenhuma rede no PROJECAO esta ausente da tabela AS:AZ
