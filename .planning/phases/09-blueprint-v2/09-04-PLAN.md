This is the largest single task in the entire CRM project. The FATURAMENTO block has 186 columns organized as:

- BZ: VENDA (standalone anchor, =CA{row} or simple reference)
- CA: % ALCANCADO (anchor for FATURAMENTO super-group)

Then 4 quarters, each containing 3 months, each with 15 sub-columns:

**Quarter structure (from research):**
- Q1: CB (anchor %Q1), then JAN(CC-CQ), FEV(CR-DF), MAR(DG-DU)
- Q2: DV (anchor %Q2), then ABR(DW-EK), MAI(EL-EZ), JUN(FA-FO)
- Q3: FP (anchor %Q3), then JUL(FQ-GE), AGO(GF-GT), SET(GU-HI)
- Q4: HJ (anchor %Q4), then OUT(HK-HY), NOV(HZ-IN), DEZ(IO-JC)

**15 sub-columns per month (in order):**
1. %YTD = cumulative year-to-date percentage = META_YTD / REAL_YTD (or REAL/META)
2. META (YTD cumulative)
3. REALIZADO (YTD cumulative)
4. %TRI = quarter percentage
5. META (quarter cumulative)
6. REALIZADO (quarter cumulative)
7. %MES = month percentage = REALIZADO_MES / META_MES
8. META MES = PROJECAO col L / 12 (annual meta / 12 per Phase 8 decision)
9. REALIZADO MES = INDEX/MATCH from DRAFT 1 monthly sales column for this month
10. DATA PEDIDO = most recent order date from DRAFT 2 within this month (array formula)
11. JUSTIFICATIVA S1 = COUNTIFS('DRAFT 2'!$D:$D,$B{row},'DRAFT 2'!$A:$A,">="&DATE(year,month,1),'DRAFT 2'!$A:$A,"<="&DATE(year,month,7))
12. JUSTIFICATIVA S2 = same pattern, days 8-14
13. JUSTIFICATIVA S3 = same pattern, days 15-21
14. JUSTIFICATIVA S4 = same pattern, days 22-end_of_month
15. JUSTIFICATIVA MENSAL = SUM(S1:S4) for this month

**Build the block programmatically:**

Create a function write_faturamento_month(ws, start_col, month_num, year, data_start_row, data_end_row, draft1_month_col, projecao_meta_col) that writes all 15 sub-columns for one month.

Loop through 12 months, calculating start_col for each. Use year=2025 for months 1-12 (or 2026 for JAN if that's the current year -- check PROJECAO data).

**Formula patterns (all with bounded ranges $3:$25000):**

META MES:
=IFERROR(INDEX('PROJECAO '!$L$4:$L$600,MATCH($B{row},'PROJECAO '!$B$4:$B$600,0))/12,0)

REALIZADO MES (varies by month -- map to correct DRAFT 1 column):
=IFERROR(INDEX('DRAFT 1'!${month_col}$3:${month_col}$25000,MATCH($B{row},'DRAFT 1'!$B$3:$B$25000,0)),0)

%MES:
=IF({meta_col}{row}=0,"",{real_col}{row}/{meta_col}{row})

META YTD (cumulative):
=SUM of all META MES from JAN to current month

REALIZADO YTD (cumulative):
=SUM of all REALIZADO MES from JAN to current month

%YTD:
=IF({meta_ytd_col}{row}=0,"",{real_ytd_col}{row}/{meta_ytd_col}{row})

Quarter aggregates: sum of 3 months within the quarter.

DATA PEDIDO:
=IFERROR(MAXIFS('DRAFT 2'!$A$3:$A$25000,'DRAFT 2'!$D$3:$D$25000,$B{row},'DRAFT 2'!$A$3:$A$25000,">="&DATE({year},{month},1),'DRAFT 2'!$A$3:$A$25000,"<="&DATE({year},{month},{end_day})),"")
(Use MAXIFS instead of complex CSE per Research Open Question #5)

JUSTIFICATIVA S1 (week 1, days 1-7):
=COUNTIFS('DRAFT 2'!$D$3:$D$25000,$B{row},'DRAFT 2'!$A$3:$A$25000,">="&DATE({year},{month},1),'DRAFT 2'!$A$3:$A$25000,"<="&DATE({year},{month},7))

S2 (days 8-14), S3 (days 15-21), S4 (days 22-end):
Same COUNTIFS pattern with different date ranges. For S4 end day: use DATE({year},{month}+1,0) which gives last day of month in Excel.

JUSTIFICATIVA MENSAL:
=SUM({s1_col}{row}:{s4_col}{row})

Quarter anchor columns (%Q1, %Q2, %Q3, %Q4):
=%MES of the last month in the quarter (or custom calculation)

CA (% ALCANCADO):
Reference the current month's %MES column.

BZ (VENDA):
=CA{row} or reference current month REALIZADO.
