```python

}
```

## Column Mapping DRAFT 1

```
Col  2: CNPJ (chave)
Cols 21-30: MAR/25 a DEZ/25 (vendas mensais)
Col 31: JAN/26
Col 33: CICLO MEDIO
Col 34: Nro COMPRAS
Col 35: CURVA ABC
Col 36: MESES POSITIVADO
Col 37: TICKET MEDIO
Col 43: MEDIA MENSAL
```

## JSON idx -> DRAFT 1 col

```
idx 0 (JAN/25): SEM COLUNA (incluir no total)
idx 1 (FEV/25): SEM COLUNA (incluir no total)
idx 2 (MAR/25): col 21
idx 3 (ABR/25): col 22
idx 4 (MAI/25): col 23
idx 5 (JUN/25): col 24
idx 6 (JUL/25): col 25
idx 7 (AGO/25): col 26
idx 8 (SET/25): col 27
idx 9 (OUT/25): col 28
idx 10 (NOV/25): col 29
idx 11 (DEZ/25): col 30
jan26_vendas[cnpj]: col 31
```

## ABC Thresholds

A >= R$ 2.000, B >= R$ 500, C < R$ 500 (sobre TOTAL de todos os 13 meses: JAN/25-JAN/26)

## CARTEIRA Formula Template (from row 7)

Vendas: `=IFERROR(INDEX('DRAFT 1'!$U:$U,MATCH($B{r},'DRAFT 1'!$B:$B,0)),"")`
Total:  `=SUM(Z{r}:AJ{r})`
ABC:    `=IFERROR(INDEX('DRAFT 1'!$AI:$AI,MATCH($B{r},'DRAFT 1'!$B:$B,0)),"")`
