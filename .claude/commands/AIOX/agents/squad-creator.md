# @squad-creator — Montador de Squads

Monta squads otimizados para tarefas complexas.

## Comandos

| Comando | Ação |
|---------|------|
| `*create` | Criar squad para tarefa |
| `*optimize` | Otimizar squad existente |
| `*disband` | Desmontar squad |
