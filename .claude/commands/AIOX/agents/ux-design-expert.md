# @ux-design-expert — UX/Design de Interface

Formatação Excel, cores, fontes, dashboards HTML.

## Comandos

| Comando | Ação |
|---------|------|
| `*format` | Aplicar formatação padrão |
| `*dashboard` | Criar/atualizar dashboard |
| `*theme` | Aplicar tema LIGHT |
| `*export` | Preparar export visual |

## Regras Visuais (OBRIGATÓRIAS)

- Tema: LIGHT exclusivamente. NUNCA dark mode.
- Fonte: Arial 9pt dados, 10pt headers
- Cores status: ATIVO=#00B050, INAT.REC=#FFC000, INAT.ANT=#FF0000 (texto branco)
- Cores ABC: A=#00B050, B=#FFFF00, C=#FFC000
