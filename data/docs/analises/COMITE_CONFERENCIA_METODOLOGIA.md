## 📋 CHECKLIST OBRIGATÓRIO

### ✅ ANTES DE RESPONDER, VERIFICAR:

#### 1. FONTE DE DADOS
- [ ] Dados vêm de arquivo específico identificado?
- [ ] Nome do arquivo está documentado?
- [ ] Linha/página da informação está citada?
- [ ] **SE NÃO:** Marcar como "ESTIMATIVA" ou "NÃO VERIFICADO"

#### 2. NÚMEROS E MÉTRICAS
- [ ] Número foi extraído diretamente da fonte?
- [ ] Cálculo matemático foi refeito e conferido?
- [ ] Há margem de erro? Está documentada?
- [ ] **SE NÃO:** Não usar o número, buscar fonte correta

#### 3. INTERPRETAÇÕES
- [ ] Interpretação tem base factual nos dados?
- [ ] Há dados contraditórios? Foram mencionados?
- [ ] Conclusão é lógica OU especulação?
- [ ] **SE ESPECULAÇÃO:** Deixar claro que é hipótese

#### 4. CITAÇÕES E FRASES
- [ ] Frase está exatamente como na fonte?
- [ ] Contexto da frase foi preservado?
- [ ] Não está parafraseando como se fosse citação literal?
- [ ] **SE NÃO:** Usar paráfrase e deixar claro

#### 5. CASOS E EXEMPLOS
- [ ] Cliente mencionado existe nos dados?
- [ ] Data do evento está correta?
- [ ] Problema descrito está documentado?
- [ ] **SE NÃO:** Remover exemplo ou buscar correto

## 🚫 PROIBIÇÕES ABSOLUTAS

### NUNCA FAZER:
1. ❌ Inventar números "razoáveis"
2. ❌ Criar exemplos fictícios "ilustrativos"
3. ❌ Inferir dados que não existem
4. ❌ Arredondar sem avisar
5. ❌ Misturar dados de fontes diferentes sem avisar
6. ❌ Assumir que padrão pequeno = padrão geral
7. ❌ Usar "provavelmente", "deve ser", "cerca de" sem base
8. ❌ Copiar análise anterior sem re-verificar fonte

## ✅ PERMISSÕES (Com Disclaimer)

```
- [ ] Arquivo 2: [nome.csv] - Coluna [Z]
- [ ] Cálculo próprio baseado em [fonte]

✅ [Dado 1]: Fonte [X], Linha [Y]
✅ [Dado 2]: Fonte [X], Linha [Z]

❌ [Dado 4]: NÃO ENCONTRADO - removido da resposta

INTERPRETAÇÕES:
✅ [Interpretação A]: Lógica porque [razão]
⚠️ [Interpretação B]: HIPÓTESE - pode ser [X] ou [Y]

LACUNAS IDENTIFICADAS:
- Gap 1: Não temos dados sobre [X]
- Gap 2: Fonte [Y] incompleta
- Gap 3: Contradição entre [A] e [B]

CONFIANÇA DA RESPOSTA:
[ ] 100% - Todos dados verificados
[ ] 80%+ - Maioria verificada, algumas estimativas
[ ] 60-79% - Mix dados + hipóteses
[ ] <60% - Predominantemente estimativas → REFAZER
```

## 🔄 PROCESSO DE VALIDAÇÃO

### ETAPA 1: COLETA (10 min)
1. Ler pergunta 3x
2. Listar fontes disponíveis
3. Anotar o que PRECISO encontrar

### ETAPA 2: EXTRAÇÃO (20 min)
1. Abrir cada fonte
2. Buscar dados específicos
3. Copiar LITERAL (não parafrasear ainda)
4. Anotar linha/página

### ETAPA 3: VERIFICAÇÃO (10 min)
1. Re-ler dados extraídos
2. Conferir cálculos
3. Identificar contradições
4. Marcar gaps

### ETAPA 4: ESTRUTURAÇÃO (15 min)
1. Organizar dados verificados
2. Separar: fatos vs hipóteses
3. Criar hierarquia confiança

### ETAPA 5: RESPOSTA (10 min)
1. Escrever resposta
2. Citar fontes
3. Explicitar estimativas
4. Documentar lacunas

### ETAPA 6: REVISÃO FINAL (5 min)
1. Reler resposta
2. Cada número tem fonte?
3. Cada conclusão tem lógica?
4. Está claro o que é fato vs opinião?

**TOTAL: ~70 min por resposta complexa**

## 🎯 REGRAS DE OURO

### 1. **"NÃO SEI" É UMA RESPOSTA VÁLIDA**
Melhor dizer "não tenho esses dados" do que inventar.

### 2. **EXPLÍCITO > IMPLÍCITO**
Sempre deixar claro o que é fato, estimativa, hipótese.

### 3. **CITE OU CALE**
Todo número precisa de fonte. Se não tem, não use.

### 4. **CONTRADIÇÃO = INVESTIGAR**
Se 2 fontes contradizem, documentar ambas, não escolher uma.

### 5. **PADRÃO ≠ CERTEZA**
"Vi 3 casos de X" ≠ "Sempre acontece X"

### 6. **VELOCIDADE < PRECISÃO**
Melhor demorar 2h e acertar que responder rápido errado.

### 7. **DOCUMENTAR TUDO**
Cada etapa, cada fonte, cada decisão.

### 8. **REVISAR ANTES DE ENVIAR**
5 minutos de revisão evitam horas de correção.

## 📊 EXEMPLO APLICADO

### ❌ MAU EXEMPLO (Sem Comitê):
> "A taxa de churn da Vitão é de 73% porque a maioria dos clientes compra apenas uma vez."

**Problemas:**
- Não cita fonte do "73%"
- "Maioria" é vago
- Não explica o cálculo

### ✅ BOM EXEMPLO (Com Comitê):
> "Analisando a Carteira Detalhada (16/Dez/2025), 349 de 477 clientes (73,2%) estão inativos. 
> 
> Cruzando com o arquivo de Vendas 2025, identificamos que 253 de 445 clientes únicos (56,9%) compraram apenas 1 vez.
> 
> **Fontes:**
> - Carteira_detalhada_de_clientes_16_01.txt (linhas 8-477)
> - Relatorio_de_Vendas_2025.txt (análise recorrência)
> 
> **Cálculo:** 349 ÷ 477 = 0,732 = 73,2%
> 
> **Limitação:** Não sabemos se todos inativos são ex-compradores ou se alguns nunca compraram."
