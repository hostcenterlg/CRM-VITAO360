### **1. CUSTO DE AQUISIÇÃO POR CLIENTE (CAC)**

**Cálculo:**
```
Salários comerciais: R$ 15k × 3 consultores = R$ 45k/mês

CAC = R$ 50.000 / 44 = R$ 1.136/cliente
```

**Interpretação:**
- Custa R$ 1.136 para adquirir 1 cliente novo
- Primeira venda precisa ser > R$ 1.136 para não ter prejuízo
- Ticket médio atual: R$ 4.155 ✅ **CAC coberto na 1ª venda**

**Benchmark indústria B2B:** R$ 800-2.000

### **3. RELAÇÃO LTV/CAC — SAÚDE DO NEGÓCIO**

**Cálculo:**
```
LTV Standard: R$ 12.400
CAC: R$ 1.136
LTV/CAC = 12.400 / 1.136 = 10,9x
```

**Interpretação:**

| Relação | Status | Significado |
|:--------|:-------|:------------|
| < 1x | 🔴 Falência | Perde dinheiro em cada cliente |
| 1-3x | 🟡 Arriscado | Margem apertada |
| 3-5x | 🟢 Saudável | Negócio viável |
| > 5x | 🟢🟢 Excelente | Margem confortável |

**Status Vitão:** **10,9x** = 🟢🟢 **EXCELENTE**

**MAS:** LTV só se realiza SE reter o cliente. Com retenção de 20%, LTV cai para:
```
LTV Real = R$ 12.400 × 0,20 = R$ 2.480
LTV/CAC Real = 2.480 / 1.136 = 2,2x
```

**Status real:** **2,2x** = 🟡 **ARRISCADO**

**Ação urgente:** **AUMENTAR RETENÇÃO** de 20% → 40%
- LTV/CAC volta para 4,4x (saudável)
- Cada 10% de aumento na retenção = +R$ 270k/ano

### **4. PAYBACK PERIOD — QUANTO TEMPO PARA RECUPERAR CAC**


### **7. SALES VELOCITY — VELOCIDADE DE VENDAS**

```
Ciclo: 5 dias

Velocity = (1.074 × 0,216 × 4.155) / 5
Velocity = R$ 192.638/dia para ativos
```
