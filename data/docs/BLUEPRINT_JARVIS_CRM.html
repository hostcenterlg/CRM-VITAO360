PLANO MULTI-AGENTES — CLAUDE CODE

Estratégia: O Master Orchestrator (Claude Opus) coordena 6 agentes especializados (Claude Sonnet), cada um responsável por uma fase do pipeline de dados. Sequência lógica baseada em dependências.
1
AGENTE EXTRACTOR — Ingestão de Dados
Lê os 7 arquivos Excel/XLS. Normaliza CNPJs (14 dígitos). Normaliza telefones (55+DDD+num). Emails lowercase. Texto uppercase sem duplo espaço. Gera arquivos intermediários JSON padronizados.
↓
2
AGENTE DEDUP — Deduplicação e Reconciliação
Cruza CNPJs entre todas as fontes. Resolve os 28 COD_SAP divergentes. Elimina 57 duplicatas do SAP SEM ATEND. Identifica 453 órfãos do LOG. Gera relatório de reconciliação.
↓
3
AGENTE BUILDER — Construção da BASE
Monta a aba BASE com 49 colunas, 1 linha por CNPJ. Aplica regras de prioridade de fonte (Mercos → SAP → Redes). Popula grupos SAP/MERCOS/META. Calcula campos derivados (SITUAÇÃO, VENDEDOR, CURVA).
↓
4
AGENTE LOG — Migração do Histórico
Migra 11.758 registros do LOG atual para o novo formato de 16 colunas. Normaliza TIPO_CONTATO (elimina notas no campo errado). Normaliza RESULTADO (PERCA→PERDA). Valida contra REGRAS.
↓
5
AGENTE REDES — Enriquecimento de Franquias
Processa as 8 abas de rede (estruturas diferentes). Vincula 777 lojas sem SAP via XLOOKUP por CNPJ. Popula campo REDE na BASE. Gera aba REDES consolidada com 899 lojas.
↓
6
AGENTE VALIDATOR — Auditoria Final
Verifica integridade referencial (todo CNPJ do LOG existe na BASE). Valida campos obrigatórios. Conta preenchimento por coluna. Gera relatório de qualidade com % de completude. Zero tolerance para fabricação.
