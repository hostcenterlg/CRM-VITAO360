2 CONCEITO: DRAFT → BASE CENTRAL → VISÕES
📋

ABA DRAFT

Cola aqui tudo que o Claude gera. Vendas, atendimentos, agendas. Formato padronizado 25 colunas.

⚙️

ABA LOG (motor)

Puxa do DRAFT via validação. Deduplicação automática. Fórmula FOLLOW-UP calcula datas.

👁️

VISÕES (filtros)

Agenda do dia, Follow-ups pendentes, Dashboard — tudo é FILTRO sobre o mesmo LOG.

Princípio: O Claude nunca edita o LOG direto. Ele gera dados na aba DRAFT. Você revisa, valida, e os dados fluem para o LOG via processo controlado.

4 FLUXO: COMO A ABA DRAFT FUNCIONA
1
Claude gera dados
Vendas, atendimentos, agenda, LOG — sempre no formato padronizado de 25 colunas
↓
2
Você cola na aba DRAFT
Zona segura — nada afeta o LOG até você validar. Pode colar múltiplos outputs seguidos.
↓
3
Indicadores visuais mostram status
🟢 Novo (não existe no LOG) | 🟡 Possível duplicata | 🔴 Duplicata confirmada
↓
4
Você revisa e seleciona
Checkbox "PROCESSAR" — marque as linhas válidas. Corrija o que precisar.
↓
5
Move para LOG (copiar + colar especial valores)
Linhas marcadas → Ctrl+C → LOG → última linha → Colar Especial. FOLLOW-UP calcula sozinha.
↓
6
Limpar DRAFT
Apagar linhas processadas. Mantém limpo para próximo batch.
Vantagem: Se o Claude errar algo, você pega na DRAFT antes de poluir o LOG. O LOG nunca é contaminado com dados não validados.

8 FILTROS E AGRUPAMENTOS NO EXCEL
📑

AGRUPAR COLUNAS

Dados > Agrupar. Oculta colunas de referência com um clique. Mostra só o essencial.

📐

AGRUPAR LINHAS

Blocos da Agenda como grupos colapsáveis. Expandir só o bloco que está trabalhando.

🔽

FILTROS AVANÇADOS

Ctrl+Shift+L. Filtrar por vendedor, data, situação, resultado, REDE. Múltiplos critérios.

Limitação Excel: Agrupamentos são manuais. A aba AGENDA será TABELA com filtro + formatação condicional. Para agrupamento dinâmico: VBA ou Google Sheets + Apps Script.
